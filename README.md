# Welcome to Webber, the dev browser for WebberBeet files, an language that is 80% Markdown and 20% HTML, written in pure python. 
How to use:
1. Pipes mean formatted text, where you can do what you need with them
2. in a formatted text box, you can use <> for bolding, :: for italizing, or <: :> for both.
3. There is also color! dirreactly after the opening |, use ** for calling a function.
4. We will use the builtin hexc() function. It takes in a hex code, which is defined by a H! starter.
